## Getting Started

### Prerequisites

* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/jimblu/MERN_CRUD-minimal-student-manager-app.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
### Use it

1. Run the app
   ```sh
   npm start
   ```
2. Start Backend Server
   ```sh
   npm install
   ```
   ```sh
   nodemon server.js
   ```
3. Database
   ```sh
   mongod
   ```
4. For production
   ```sh
   npm run build
   ```
